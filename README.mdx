# JSON Key Suggestion

This VS Code extension provides autocomplete suggestions for JSON keys based on a user-defined JSON file.

## Features

- Auto-complete suggestions for JSON keys in `.mdx` files.
- Dynamically loads keys from a user-specified JSON file.

## Configuration

Add the following setting in your `settings.json` file to specify the JSON file path:

```json
"jsonKeySuggestion.jsonFilePath": "./data/translations.json"
